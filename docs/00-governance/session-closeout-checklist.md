# 会话收尾清单

重要工作结束前按这个顺序检查：

1. 本次目标是否完成，未完成项是否有明确原因。
2. 相关文档、决策或账本是否已更新。
3. 是否避免写入密钥、登录态、二维码、隐私截图和未脱敏资料。
4. 是否运行了本地治理检查。
5. 是否同步并验证了个人知识库镜像。
6. 是否检查了 Git 状态。
7. 如已授权，是否完成提交和推送。
8. 汇报是否包含改动、验证、Git 状态、知识库状态和下一步。

推荐命令：

```powershell
powershell -ExecutionPolicy Bypass -File tools/run-closeout.ps1 -Summary "本次完成的事情"
```

如果涉及金融、量化或投研内容，汇报中必须保留边界：历史模拟和研究辅助不等于投资建议、荐股或收益承诺。
