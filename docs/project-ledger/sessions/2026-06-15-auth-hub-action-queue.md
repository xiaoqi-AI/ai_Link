# 2026-06-15 授权中枢待处理队列展示

## 本次推进

- `GET /api/tasks` 支持 `status` 查询参数，用于筛选 `action_required`、`queued` 等任务。
- 控制台首页新增状态计数，突出排队、执行中、待人工处理和待确认发布。
- 控制台首页新增“待人工处理”表格，真实连接器遇到续登、验证码或限流后可以直接从这里进入任务详情。

## 价值

- 真实微信/朱雀连接器接入后，人工处理事项不再只能混在普通任务列表里找。
- 本地执行器把任务停到 `action_required` 后，控制台能形成“发现问题 -> 人工处理 -> retry 重新排队”的闭环。

## 安全边界

- 列表只展示脱敏摘要和任务状态。
- 不展示 Cookie、二维码、浏览器 Profile、截图或平台原始页面。
