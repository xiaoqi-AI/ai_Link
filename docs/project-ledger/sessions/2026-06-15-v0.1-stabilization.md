# 2026-06-15 v0.1 稳定化

## 目标

继续推进 AI Link v0.1 稳定化，让公开仓更接近外部用户可复现状态。

## 已补充

- GitHub Actions CI：安装依赖、类型检查、测试、配置校验、安全扫描和生产依赖审计。
- `ai-link config validate`：校验 provider、route、policy 和常见配置风险。
- `tools/verify-fresh-clone.js`：模拟 fresh clone 后重新安装、检查、测试、配置校验、CLI dry-run 和安全扫描。
- 文档补充：README、用户指引、配置说明和开放问题。

## 验证重点

- `npm run check`
- `npm test`
- `npm run ai-link -- config validate`
- `npm run security:scan`
- `npm run verify:fresh`
- 治理检查和知识库镜像校验

## 边界

- 真实外部 provider 调用仍需要用户本机密钥，不在公开仓验证。
- `auth-hub` 已进入公开骨架，但真实平台连接器仍需私有实现和人工授权边界。
